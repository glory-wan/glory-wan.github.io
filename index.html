研究兴趣：

          0 我感兴趣的研究方向是： 类脑智能、人工智能、知识表征学习、脑科学、认知神经科学。
          正如现在AI热潮的趋势所向，个人认为AI未来的发展方向必然是多模态数据/模型的大统一。 正如一个真正的人一样，一个模型可以在多种任务上，而不只是单一的在某个方向上表现出色。
          1 对于专业之外，我也对量子力学感兴趣。
          物理是探索我们这个世界的最直接工具，探究物理就是探究宇宙的逻辑底层运行规律（我很喜欢探究事物背后的原因，无论自然科学或者人文科学）。 其次，我认为想要做到真正的类脑智能，不能仅仅靠现在的深度学习的方式（通过堆积大量的人工神经元以及增加网络层深度和宽度来拟合目标函数）来实现。 另一方面，从数据计算方式看，我认为量子位的多种状态叠加可能更加贴合人脑计算方式。
          2 和上述同样的原因，我对社会学与心理学同样存在浓厚的兴趣。
          社会学和心理学都是与人紧密相关的学科，社会学从宏观的角度观察和解释人群体的行为与现象，心理学从微观角度研究每一个独立的人类个体。 学习社会学和心理学知识，运用到我们日常生活中，从底层逻辑分析人们各种行为和现象， 可以让我们深刻全面的认识自己与我们所处的社会。

          当我成为一名独立的PI之后，基于量子计算的智能体将是我的研究方向：
          研究出真正意义上的“类脑智能”，或者说“强人工智能”

开源项目

LLIE Library

一个集成了多种传统低照度增强方法的 Python 库
基于Pytorch编写的一个可灵活拓展的集成关于传统低照度增强方法的Python库。

  • 目前支持算法有：HE系列(HE/CLAHE/RCLAHE)、gamma矫正、log变换、DarkChannel...
  • 目前支持的颜色空间有：RGB、HSV、HLS、LAB、YUV

代码连接：https://github.com/glory-wan/LLIE-Lib
LLIE Website

基于上述LLIE Lib开发的低照度图像增强算法平台
利用java后端和VUE提供可视化的低照度图像增强算法使用界面，并利用阿里云服务器部署上线。

代码连接：https://github.com/glory-wan/LLIE-web

网站链接：http://iaplatform.cn
face detection and tracking

利用Dlib和haar级联分类器实现人脸/物体的识别与跟踪

  • 支持的模型有：frontal_face_detector、face_landmarks、haarcascade系列...

代码连接： https://github.com/glory-wan/face-detection-and-tracking
Tobacco Recognition

基于YOLOv8与Django实现的云端烟草识别接口
使用YOLOv8模型训练烟草数据集，联合使用Django框架和Ultralytics的API实现烟草识别接口，最后利用阿里云ESC服务器部署到云端。

  • 现在已经被重庆烟草局总公司接收，展开实际商业合作
